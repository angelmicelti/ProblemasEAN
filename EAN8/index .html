Ejercicio 1: Serie y Paralelo

Acabamos de comprar dos condensadores, uno de 40 µF y otro de 60 µF.
1. ¿A cuánto equivalen si los unimos consecutivamente (serie)?
2. ¿Y si conectamos los terminales de la izquierda entre sí y los de la derecha entre sí (paralelo)?

Caso 1: Conexión en Serie ("consecutivamente")

En serie, la capacidad total disminuye (es como aumentar la distancia entre placas). La fórmula es la inversa de la suma de las inversas:

1/Ceq = 1/C1 + 1/C2

O para dos condensadores:

Ceq = (C1 · C2) / (C1 + C2)
Ceq = (40 · 60) / (40 + 60) = 2400 / 100 = 24 µF

Caso 2: Conexión en Paralelo

En paralelo, las capacidades se suman directamente (es como aumentar el área de las placas):

Ceq = C1 + C2
Ceq = 40 + 60 = 100 µF

Resultados:

Serie: 24 µF

Paralelo: 100 µF

Ejercicio 3: Asociaciones en Serie

Calcula la capacidad equivalente en serie. Recuerda la fórmula general: 1/CT = 1/C1 + 1/C2 + ...

Asociación 3.1

200µF -- 300µF

Asociación 3.2

200nF -- 250nF -- 300nF

Asociación 3.3

125µF -- 300µF -- 3mF

Asociación 3.1 (Dos condensadores)

CT = (C1 · C2) / (C1 + C2)
CT = (200 · 300) / (200 + 300) = 60000 / 500 = 120 µF

Asociación 3.2 (Tres condensadores)

1/CT = 1/200 + 1/250 + 1/300
1/CT = 0.005 + 0.004 + 0.00333... = 0.01233...
CT = 1 / 0.01233... = 81.08 nF

Asociación 3.3 (Cuidado con las unidades)

Convertimos todo a µF: 3 mF = 3000 µF

1/CT = 1/125 + 1/300 + 1/3000
1/CT = 0.008 + 0.00333 + 0.00033 = 0.01166...
CT = 1 / 0.01166... = 85.71 µF

Soluciones:

3.1: 120 µF

3.2: 81.08 nF

3.3: 85.71 µF

Ejercicio 5: Tiempo de Carga

Calcula cuánto tiempo tardará en cargarse el condensador del montaje.
Se considera que un condensador está cargado aproximadamente en 5 · τ (5 veces la constante de tiempo).

Datos iniciales visuales: R = 10 kΩ, C = 200 µF

Caso 1: Datos originales

R = 10 kΩ = 10,000 Ω
C = 200 µF = 0.0002 F

τ (Tau) = R · C
τ = 10,000 · 0.0002 = 2 segundos.
Tiempo total (5τ) = 5 · 2s = 10 s

Caso 2: Cambiando la resistencia

Nueva R = 4k7 = 4,700 Ω
C = 200 µF

τ = 4,700 · 0.0002 = 0.94 segundos.
Tiempo total (5τ) = 5 · 0.94s = 4.7 s

Caso 3: Cambiando R y C

R = 4k7 = 4,700 Ω
Nuevo C = 100 µF = 0.0001 F

τ = 4,700 · 0.0001 = 0.47 segundos.
Tiempo total (5τ) = 5 · 0.47s = 2.35 s

Tiempos de carga total:

t1 = 10 s

t2 = 4.7 s

t3 = 2.35 s

Ejercicio 6: Diseño de Condensadores

Desafío de lógica de circuitos:
1. ¿Cómo construir un condensador de 10 µF teniendo solo condensadores de 30 µF?
2. ¿Y a partir de condensadores de 100 µF?

Caso 1: Objetivo 10µF usando piezas de 30µF

Sabemos que al poner condensadores iguales en serie, la capacidad se divide por el número de condensadores (C_eq = C / n).

Queremos pasar de 30 a 10. Dividimos por 3.

30µF / 3 = 10µF
Solución: Conectar 3 condensadores de 30µF en serie.

Caso 2: Objetivo 10µF usando piezas de 100µF

Mismo principio. Queremos pasar de 100 a 10. Necesitamos dividir por 10.

100µF / 10 = 10µF
Solución: Conectar 10 condensadores de 100µF en serie.
